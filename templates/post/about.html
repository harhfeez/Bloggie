About Bloggie

Bloggie is a blog app created by Fasasi Hafis a young promising Software Developer using Django framework.
Bloggie is a unique app where registered user can easily write blog and publish it for the world to access. As a registered user you have access to create new post, like and comment on every post published on the app. User can edit and delete post created by the user
After a user create new post, it will automatically direct to draft page where only the user can see the post before publishing it.
